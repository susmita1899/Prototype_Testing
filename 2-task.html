What you'll do

In this study, you will interact with a prototype interface designed to support healthcare professionals in analyzing patient data. You will be asked to complete two short tasks:

  • Task 1: Build a prediction pipeline. Use the interface to create a simple pipeline that predicts the likelihood of a patient outcome (e.g., “risk of complication”). Configure the pipeline by selecting input variables (like age, test results) and a model type. Run the pipeline and view the prediction result.
  • Task 2:Analyze patient trend data. Use the interface to analyze patient cohort trends (e.g., “blood pressure development over time”). Adjust parameters (e.g., age group, treatment type) to refine the trend analysis. Interpret the visualized trend output.
Tip: If you get stuck, note the time and continue. Assistance policies differ across tasks.

Consent & Recording

Please complete each task to the best of your ability. There are no right or wrong answers—the goal is to evaluate how the interface supports you while performing these tasks. If you are unsure about a step, try to continue in the way that feels most natural to you.
